# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of Torarr seriously. If you discover a security vulnerability, please follow these guidelines:

1. **Do not open a public issue** on GitHub.
2. Please report security vulnerabilities by emailing the maintainer [security@ericslutz.dev](mailto:security@ericslutz.dev).
3. Include as much information as possible to help us reproduce the issue.

We will make every effort to acknowledge your report within 48 hours and provide an estimated timeframe for a fix.
